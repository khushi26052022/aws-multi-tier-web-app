# AWS Multi-Tier Web Application

## Project Overview
Designed and deployed a highly available multi-tier web application on AWS using core infrastructure services.

## AWS Services Used
- Amazon EC2
- Amazon VPC
- Amazon RDS (MySQL)
- Application Load Balancer
- Auto Scaling
- IAM
- CloudWatch

## Architecture
Internet → Application Load Balancer → EC2 Instances → Amazon RDS

## Features
- High Availability
- Auto Scaling
- Secure Networking using VPC
- Database hosted on Amazon RDS
- Infrastructure Monitoring with CloudWatch

## Outcome
Successfully deployed a scalable and highly available web application on AWS.
